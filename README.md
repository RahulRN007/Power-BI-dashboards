## 📖 Overview

### Uber Auto Performance Dashboard
- **File:** `Power BI/Uber.pbix`
- **Interactive Highlights:**
  - KPIs such as **Completed Bookings**, **Lost Bookings**, and **Revenue** update dynamically when a vehicle type is selected.
  - Explore performance by vehicle categories: Auto, Bike, Go Mini, Sedan, Premier, XL.
  - Drill down into **driver ratings** and **average trip distance** for each vehicle.
  - Monthly revenue breakdown adjusts based on your selection, giving a clear view of trends per category.
